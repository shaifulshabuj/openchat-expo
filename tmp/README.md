# Temporary Files

All temporary files, planning documents, session notes, etc. go here.
This directory is gitignored - nothing here will be committed.

## Usage:
- Task decomposition: tmp/decomposition_YYMMDDHHMMSS.md
- Task output: tmp/output_YYMMDDHHMMSS.md
- Verification logs: tmp/verification_YYMMDDHHMMSS.md
- Error logs: tmp/error_YYMMDDHHMMSS.md
- Session notes: tmp/session_YYMMDDHHMMSS.md
- Session summaries: tmp/session_summary_YYMMDDHHMMSS.md

## Cleanup:
Clean up old files periodically to save space.
